# Holy Quran Teacher App

A cross-platform Flutter application designed as a landing page for a Holy Quran teaching service. It features an introductory video placeholder and prominently displays contact information for users who want to learn Tajweed.

## Features
- **Video Introduction**: A placeholder for an AI-generated or recorded introductory video from the teacher.
- **Service Description**: Details about learning to recite the Holy Quran with proper Tajweed.
- **Contact Action**: A clear call-to-action displaying the contact number (033242447800) for scheduling classes.
- **Responsive Design**: Works perfectly across mobile, tablet, and desktop screens.

## Getting Started

### Prerequisites
- Flutter SDK (latest stable version)
- Dart SDK

### Run the App
1. Clone the repository
2. Run `flutter pub get` to install dependencies
3. Run `flutter run` to start the application
